A demonstrative website for event meetups. Functioning backend in NodeJS with minimalist frontend.

Derangement - Game Prototype

<

>

The game prototype I created in Unity 5 for Advanced Game Design and Development

1. For this project, I wanted to do something somewhat, but not overtly, scary. The theme I settled on was a murder mystery set in a strange town. The design would incorporate mild horror elements to bring atmosphere to a Metroidvania-style RPG adventure game where the player could make choices that impacted the narrative and progression.

2. Most of the assets in this project were from publicly available resources (not without citing & mild tweaking). I also wanted used 2d sprites for the character models to add surrealism. The dialogue function and interactivity was programmed from scratch and utilizes Unity's textbox interface for simplicity.

3. Prior to creating the game, I developed a fully written story and game map that functioned as a rough overview. This prototype serves as the opening area of the game, and has multiple environments such as the one shown here. I also relied on Unity's rendering effects to create dark and foggy scenery, and created scene transitions such as moving from day to night.
